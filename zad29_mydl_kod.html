1. W jaki sposób można określić właściwość danego obiektu (dwa sposoby wraz z przykładami)?

Można określić właściwość obiektu poprzez:

  • Notację kropkową: obiekt.właściwość = "wartość"; np. samochod.kolor = "czerwony";
  • Notację nawiasów kwadratowych: obiekt["właściwość"] = "wartość"; np. samochod["kolor"] = "czerwony";

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

Można odwołać się do pól i metod obiektu używając notacji kropkowej, np. obiekt.pole lub obiekt.metoda(), a także notacji nawiasów kwadratowych, np. obiekt["pole"].

3. Wymień cechy charakterystyczne konstruktora.

  • Ma taką samą nazwę jak klasa.
  • Nie zwraca wartości (nawet void).
  • Jest wywoływany automatycznie przy tworzeniu obiektu.

4. Konstruowanie obiektu na podstawie przykładu - obiekt ma trzy pola oraz jedną metodę.

class Samochod {
    constructor(marka, model, rok) {
        this.marka = marka;
        this.model = model;
        this.rok = rok;
    }
    wyswietlInfo() {
        return `${this.marka} ${this.model} (${this.rok})`;
    }
}
let auto = new Samochod("Toyota", "Corolla", 2020);
console.log(auto.wyswietlInfo());

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

this odnosi się do aktualnie tworzonego obiektu i pozwala na dostęp do jego pól i metod. Pola oraz metody oddzielamy za pomocą kropki (.), np. this.pole.

6. Konstruktor w JS na podstawie przykładu wraz z opisem.

function Osoba(imie, wiek) {
    this.imie = imie;
    this.wiek = wiek;
    this.przedstawSie = function() {
        return `Cześć, jestem ${this.imie} i mam ${this.wiek} lat.`;
    };
}
let osoba1 = new Osoba("Jan", 25);
console.log(osoba1.przedstawSie());

Konstruktor to specjalna funkcja, która tworzy nowe obiekty i przypisuje im wartości pól.

7. Znaczenie słowa kluczowego New wraz z przykładem.

Słowo kluczowe new służy do tworzenia nowych instancji obiektów na podstawie funkcji konstruktora.

let auto = new Samochod("Ford", "Focus", 2018);

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem.

prototype umożliwia dodawanie metod do wszystkich instancji danej klasy.

function Zwierze(gatunek) {
    this.gatunek = gatunek;
}
Zwierze.prototype.dajGlos = function() {
    return `${this.gatunek} wydaje dźwięk.`;
};
let pies = new Zwierze("Pies");
console.log(pies.dajGlos());

Prototype pozwala na rozszerzanie właściwości i metod obiektów, co oszczędza pamięć.
